# Todo App

## Features
- Can add Todo task
- Delete Todo task
- Get all tasks even after leaving browser and coming back.
- Not used local storage but vercel db to store data

## Stack
- NextJs
- TypeScript
- Tailwind
- Drizzle ORM
- Vercel Postgres SQL
  
[See Live](https://todo-next-drizzle-sanity.vercel.app/)
